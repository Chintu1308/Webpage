Activity 9: Create a website using GitHub

  1. Sign in to your GitHub account at github.com. If you don't have an account, click on "Sign up" to create one.
  2. Once signed in, click on the "+" icon at the top right corner of the page and select "New repository" from the dropdown menu.
  3. In the "Repository name" field, enter a name for your repository, such as "my-website".
  4. (Optional) In the "Description" field, provide a brief description for your repository.
  5. Make sure the repository visibility is set to "Public" to allow others to access your website.
  6. (Optional) Check the "Initialize this repository with a README" option to create a README file for your repository.
  7. Click on the button to create your repository.
  8. Navigate to your newly created repository on GitHub, where you want to host your website.
  9. Click on the "Add file" button located near the top right corner of the repository interface. From the dropdown menu, select "Upload files".
  10. You can either drag and drop your HTML file and image file directly into the file upload area or click on "Choose your files" to select them from your computer.
  11. After uploading the files, GitHub will display them in the file list. Below the file list, you'll find a text box where you can enter a commit message. This message describes the changes you're making to the repository. For example, you can write "Initial commit" to signify that this is the first version of your website.
  12. Once you've entered the commit message, click on the green "Commit changes" button to save your files to the repository.
  13. Now that your files are in the repository, it's time to set up GitHub Pages to host your website.
  14. In your repository, click on the "Settings" tab located near the top right corner of the repository interface.
  15. Scroll down to the Pages within the Settings page.
  16. Under the "Source" section, you'll see a dropdown menu. Select the branch you want to use for your GitHub Pages deployment. This is typically the "main" or "master" branch where your HTML file resides.
  17. After selecting the branch, click on the Save button to apply the changes.
  18. GitHub will now build your website using the files from the selected branch. Once the build process is complete, GitHub will provide you with a URL to access your website.
  19. The URL will typically be in the format: https://[your-username].github.io/[repository-name]/. You can click on the provided link or copy and paste it into your web browser to view your website.
  20. Whenever you want to update the content of your website, simply navigate back to your repository on GitHub.
  21. Click on the file you want to edit, such as your HTML file.
  22. Click the pencil icon (Edit) to make changes to the file.
  23. After making your changes, scroll down to the bottom of the page and provide a commit message describing the updates you've made.
  24. Finally, click on the green "Commit changes" button to save your edits.
  25. GitHub will automatically rebuild your website with the updated content. You can then revisit the URL to see the changes reflected on your live website.

Activity 16: Create a local git repository

  1. Open the Git Bash application on your machine.
  2. Using the navigation commands direct your path to the directory where you want to create the local git repo.
  3. Then after entering the path, initiate the git local repo by using the Unix command git init.
  4. Then a hidden directory is created in your current directory which can be seen by running the command ls -ah (You will find the ".git" directory as hidden content).
  5. To add all the files at once to the git repository, run the following git command: git add .
  6. To check whether the files are added or not, type the command git status.

    Activity 20: Install Python

    1. Method 1: From Microsoft Store
      1. Open any terminal and run "python" in it. (This works even in the VS Code terminal)
      2. A Microsoft Store window opens with a search result of all Python versions. Open the desired version you want to download.
      3. Click on "Get", then click on "Install".
      4. Once the download is completed, you can find it in the "All apps" list.
    2. Method 2: From python.org
      1. Visit the official Python website: https://www.python.org/
      2. Click on the "Downloads" tab in the top navigation menu.
      3. Scroll down to find the latest version of Python. Click on the "Download Python 3.12.1" button (where 3.12.1 is the version number).
      4. Once the installer is downloaded, double-click on the downloaded file (python-3.12.1.exe) to run the installer.
      5. On the first page of the installer, make sure to check the box that says "Add Python 3.12 to PATH" before proceeding. This makes it easier to run Python from the command line. Click "Install Now" to begin the installation.
      6. The installer will now copy the necessary files and install Python. You can see the progress on the screen.
      7. Once the installation is complete, you'll see a screen with the message "Setup was successful." Make sure to check the box that says "Disable path length limit" if it appears.
      8. Open the Command Prompt and type "python --version" or "python -V" to verify that Python is installed. You should see the Python version number.
